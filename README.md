# User_Registration_and_Login_System

This project is a User Registration and Login System developed using HTML, CSS, and JavaScript. It allows users to create an account, log in securely, and store user information using browser local storage. The system includes input validation, an admin dashboard to view and delete users, and basic authentication features. The application demonstrates front-end web development concepts and user management functionality in a simple and interactive interface.
